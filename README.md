# My Pocket Option Signal Bot

Telegram bot that sends BUY/SELL signals for OTC pairs with timeframes (5s/10s/15s/30s).

## Features
- Admin-only access
- Inline buttons for quick signal creation
- Timeframe selection
- Ready for real Pocket Option trading (add WebSocket logic)

## Deploy on Railway

1. Push this repo to GitHub
2. Go to [railway.app](https://railway.app) → New Project → Deploy from GitHub
3. Add environment variables from `.env`
4. Deploy

The bot will run 24/7.
