# 🚀 Pocket Option OTC Signal Bot

A **real-time Telegram signal bot** for Pocket Option OTC markets.

### ✨ Features
- Live BUY/SELL signals on OTC pairs
- Beautiful signals with emojis
- Interactive Asset Selection Menu (✅ / ⬜)
- Real candle data via WebSocket
- Deployable on Railway

### ⚠️ Warnings
- Trading involves high risk of loss.
- Use on **Demo** first.
- Never share your `POCKET_SSID`.

### Files
- `main.py` - Main bot
- `config.py` - Assets list
- `strategy.py` - EMA + RSI strategy
- `telegram_handler.py` - Signals & menu
- `.env` - Credentials

Made for educational purposes.
